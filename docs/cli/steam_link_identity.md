## steam link identity

Link Identity

### Synopsis


Link Identity

Examples:

    Link an identity with a workgroup
    $ steam link identity --with-workgroup \
        --identity-id=? \
        --workgroup-id=?

    Link an identity with a role
    $ steam link identity --with-role \
        --identity-id=? \
        --role-id=?

```
steam link identity [?]
```
